> **版本**：V1.0

| 一级模块 | 功能数量 | 核心能力 |

| 4. 知识复用与智能 | 16项 | 模板库、跨图引用、AI辅助 |

| 6. 绩效分析 | 8项 | 数据统计、报表导出 |
